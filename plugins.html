Plugins

Get specialized analysis and additional data sources

Installing them is easy!

The easiest way to install plugins is directly from within Geneious by selecting Tools → Plugins from the menu.

Make your own

You can write your own plugins for Geneious including viewers, databases, sequence operations and more using our free Java Plugin Development Kit

Top 10 Plugins

  • Microsatellites Streamlined microsatellite genotyping. Call peaks, bin them and produce a table of genotypes to export for further analysis.
  • PlasmoDB and PiroplasmaDB Search and download gene and protein information on eukaryotic pathogens.
  • QuRE Viral quasispecies reconstruction from next-generation sequencing data.
  • DeCypher Use our simple graphical interface to run batch sequence searches on your TimeLogic DeCypher server from within Geneious. Requires a DeCypher accelerator card.
  • GenomeQuest Geneious now integrates with GenomeQuest! Requires a GenomeQuest account.
  • Genbank Submission Geneious now supports uploading your sequences directly to GenBank. Simplify the process of submitting sequences, genomes, features, primers, traces, and more!
  • PAUP* Run PAUP* phylogenetic inference from inside Geneious. You will need a licensed copy of PAUP* to use this plugin. Includes Modeltest (GPL).
  • MrBayes Runs the well known program MrBayes (included) for Bayesian estimation of phylogeny. Includes interactive display of posterior output.
  • DualBrothers Recombination Detection Detect recombination and spatial phylogenetic variation using the freely available program DualBrothers.
  • PhyML Maximum Likelihood tree building for alignments.
